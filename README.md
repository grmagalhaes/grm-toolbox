# grm-toolbox
A toolbox for Java including classes for
  compress: zip and 7zip formats
	crypto: sha-1, sha-256, sha-512, MD5 and AES functions
	db: update, delete, select using jdbc
	io: Base64, FileSystem, FileTye, IniFile and Property additional functions
	json: create and read json files
	math: special random function
	msoffice: read Excel files
	net: many utils for HTML, FTP, FTPS, LDAP, RPC, SMTP and Socket
	rss: read and create rss files
	util: DateTime, HTMLExcape, I18n, Logger, OS and Text additional functions
